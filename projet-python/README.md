# Projet Python  
Ce projet a pour objectif d'analyser des indicateurs socio-économiques avec les données de la Banque mondiale

## Fonctionnalités principales
1. Récupération des données :
   Téléchargement des données des indicateurs via l'API de la Banque mondiale.

2. Nettoyage et structuration :
   Gestion des données manquantes et restructuration des données sous un format analysable.

3. Visualisation et analyse :
   - Graphiques de tendances temporelles par région et indicateur.
   - Cartes de corrélation entre les indicateurs.

4. Modélisation avancée :
   - Analyse en composantes principales (ACP) pour visualiser les similarités entre pays.
   - Clustering (K-means) pour identifier des groupes de pays aux caractéristiques similaires.
